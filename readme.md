# Creating search request

Example of using an LLM to create search API payload for Azure Search from a user prompt

Copied from a [larger repo](https://github.com/Azure/azure-search-vector-samples/blob/main/demo-python/code/indexers/csv.ipynb) and busy adding features.

### Setup
1. Clone to local or open in Codespace
2. Copy env-sample to .env
3. Update .env with your settings (blob storage, GPT and embedding model)
3. Open csv.ipynb (jupyter) and follow instructions in it


**Note** that .env is .gitignor'ed to protect your secrets.
